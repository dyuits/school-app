📋 Windows 설치 설명서

🔔 교실 알림 시스템

교무실에서 학생을 호출하면 교실 전자칠판에 팝업 알림과 호출음이 즉시 표시되는 실시간 알림 시스템입니다.

🪟 Windows 7 이상 🌐 Google Chrome 필요 📶 인터넷 연결 필요 ⚡ 설치 시간: 1분 이내
🏠 개요 ✅ 요구사항 🚀 설치 방법 ⚙️ 동작 방식 🔄 자동 실행 🗑️ 제거 방법 🔧 문제 해결 ⚠️ 주의사항
1

개요 — 교실 알림 시스템이란?

교무실에서 특정 학생을 호출하면 해당 학생의 교실 전자칠판 PC에 전체 화면 팝업 창과 호출음·TTS 음성 안내가 즉시 표시됩니다. Chrome은 평소에 화면에 보이지 않게 숨겨져 있으며, 호출이 올 때만 팝업이 나타납니다.

🙈
Chrome 백그라운드 상주
평소에는 화면 밖에 숨겨져 있어 수업에 방해되지 않음
📺
전체 화면 팝업 알림
호출 시 팝업 창이 전면에 나타나고 호출이 끝나면 자동으로 닫힘
🔊
호출음 + TTS 음성
알림음과 함께 학생 이름을 음성으로 읽어 즉시 인지 가능
🛡️
VBS 감시자(Watchdog)
Chrome이 강제 종료되면 3~15초 내에 자동으로 재시작
2

요구사항

🪟
Windows 7 이상 Windows 7 / 8 / 10 / 11 모두 지원됩니다. VBScript(VBS)가 기본 내장되어 있어 별도 설치가 필요 없습니다.
🌐
Google Chrome 브라우저 Chrome이 설치되어 있어야 합니다. 없으면 google.com/chrome 에서 무료로 설치하세요.
📶
인터넷 연결 Firebase 실시간 통신을 사용합니다. 교내 네트워크(유선/무선)에 연결되어 있어야 합니다.
🔑
관리자 권한 불필요 일반 사용자 권한으로 설치 가능합니다. 시작 프로그램은 현재 로그인 계정에만 등록됩니다.
3

설치 방법

1

autostart 폴더를 USB에 복사

이 설명서가 들어 있는 autostart 폴더 전체를 USB 드라이브에 복사한 뒤, 설치할 교실 PC에 USB를 꽂습니다.

2

해당 반의 설치 파일 더블클릭

학년·반에 맞는 📄 설치_X-Y.bat 파일을 더블클릭합니다. (파일이 30개 준비되어 있습니다)

1학년 3반 → 설치_1-3.bat
2학년 7반 → 설치_2-7.bat
3학년 10반 → 설치_3-10.bat
3

설치 완료 확인

검은 명령 창이 잠깐 나타났다 닫히고, 작업표시줄에 작은 아이콘(텍스트 없음)이 표시되면 설치 완료입니다.

Chrome 창은 화면 밖에 숨겨져 있어 보이지 않는 것이 정상입니다. 아이콘만 확인하면 됩니다.

💡
설치 시 자동으로 처리되는 내용
  • Chrome을 화면 밖(오프스크린)에서 숨김 모드로 실행
  • 작업표시줄 아이콘만 표시(텍스트 숨김) — 레지스트리 자동 설정
  • Windows 시작 프로그램 폴더에 VBS 파일 등록 (재부팅 시 자동 실행)
⚠️
작업표시줄에 텍스트가 보이면?

레지스트리 설정은 재부팅 후에 적용됩니다. 설치 직후에는 텍스트가 보일 수 있으며, PC를 한 번 재시작하면 아이콘만 표시됩니다.

4

동작 방식

평소엔 Chrome이 완전히 숨겨져 있다가, 호출이 오면 팝업만 나타나고 끝나면 자동으로 닫힙니다.

🙈
Chrome 숨김 상태로 상주

Chrome은 화면 밖(오프스크린, 좌표 -10000, 0)에서 실행됩니다. 화면에는 보이지 않고 작업표시줄에 작은 아이콘만 표시됩니다. Firebase에 연결하여 호출 신호를 대기합니다.

📢
교무실에서 호출 발생

교무실 시스템에서 학생 호출 신호를 Firebase에 전송합니다.

📺
팝업 창 전체 화면 표시

Chrome이 신호를 수신하면 별도 팝업 창(call.html)을 전체 화면으로 즉시 엽니다. 기존 숨겨진 Chrome 창은 그대로 유지됩니다.

🔊
호출음 + TTS 음성 재생

팝업 창에서 알림음과 학생 이름 TTS 음성이 재생됩니다.

✅
팝업 자동 닫힘

호출이 종료되면 팝업 창이 자동으로 닫히고, 숨겨진 Chrome은 다시 대기 상태로 돌아갑니다.

🙈
다시 숨김 상태로 복귀

화면에는 아무것도 보이지 않으며 다음 호출을 대기합니다.

5

자동 실행

🔄
PC 재부팅 시 자동 시작 Windows 시작 프로그램 폴더에 VBS 파일이 등록되어, PC를 켤 때마다 자동으로 실행됩니다.
🛡️
VBS 감시자(Watchdog) 상주 VBS 스크립트가 Chrome 프로세스를 주기적으로 감시합니다. Chrome이 강제 종료되면 3~15초 이내에 자동으로 재시작합니다.
🔒
중복 실행 방지 Lock 파일 방식으로 VBS가 이미 실행 중이면 추가 인스턴스가 생성되지 않습니다.
🙈
항상 숨겨진 상태 유지 재시작 후에도 Chrome은 화면 밖에서 실행됩니다. 수업 중 화면을 방해하지 않습니다.
⚠️
Chrome을 강제 종료해도 자동 복구됩니다

작업 관리자 등으로 Chrome을 종료해도 VBS 감시자가 3~15초 내에 자동으로 재시작합니다. 완전히 종료하려면 제거_X-Y.bat 를 실행하세요.

6

제거 방법

1

해당 반의 제거 파일 더블클릭

설치한 반에 맞는 🗑️ 제거_X-Y.bat 파일을 더블클릭합니다.

1학년 3반 → 제거_1-3.bat
2학년 7반 → 제거_2-7.bat
2

자동으로 처리되는 내용

✖ VBS 감시자 프로세스 종료 (Lock 파일 제거)
✖ 실행 중인 교실 알림 Chrome 창 종료
✖ Windows 시작 프로그램 폴더에서 등록 해제
✖ 작업표시줄 텍스트 숨김 레지스트리 설정 원복
ℹ️
재설치 시

문제가 생겼을 경우 제거 bat → 설치 bat 순서로 재설치하면 대부분의 문제가 해결됩니다.

7

문제 해결

Q
Chrome이 없다고 나오거나 실행이 안 돼요
Google Chrome이 설치되어 있지 않습니다.
google.com/chrome 에서 Chrome을 설치한 뒤 설치 bat 파일을 다시 실행하세요.
Q
아이콘이 여러 개 생겨요 (중복 아이콘)
설치 bat를 여러 번 실행하면 중복될 수 있습니다.
제거 bat를 먼저 실행하여 기존 설치를 모두 정리하세요.
그 다음 설치 bat를 한 번만 실행하세요.
Q
호출이 왔는데 소리가 나지 않아요
팝업 창(call.html)이 소리를 담당합니다. 브라우저 자동 재생 정책으로 인해 화면을 한 번 클릭해야 첫 소리가 활성화될 수 있습니다.
PC 볼륨이 음소거되지 않았는지 확인하세요.
Chrome 탭에 🔇 아이콘이 표시되어 있으면 클릭하여 음소거를 해제하세요.
Q
작업표시줄에 텍스트가 계속 보여요
레지스트리 설정은 재부팅 후에 적용됩니다.
설치 후 PC를 한 번 재시작하면 아이콘만 표시됩니다.
Q
인터넷 연결이 끊겼어요 / 오프라인 표시가 나와요
교내 Wi-Fi 또는 유선 LAN 연결 상태를 확인하세요.
인터넷이 복구되면 자동으로 재연결됩니다. 별도 조작이 필요 없습니다.
Q
재부팅 후 자동 실행이 안 돼요
설치 bat를 한 번 더 실행하면 시작 프로그램이 다시 등록됩니다.
로그인 계정이 바뀐 경우, 새 계정에서 설치 bat를 실행해야 합니다. (계정별로 등록됨)
Q
보안 경고(SmartScreen) 창이 나와요
Windows Defender SmartScreen 경고가 나타나면 "추가 정보" → "실행"을 클릭하세요.
이 시스템은 학교 내부용 스크립트이며 안전합니다.
8

주의사항

🏷️
한 PC에 하나의 반만 설치

동일 PC에 여러 반을 설치하면 충돌이 발생합니다. 반드시 하나의 반만 설치하세요.

🔄
반 변경 시 제거 후 재설치

다른 반으로 변경할 때는 기존 반의 제거 bat 먼저 실행 후, 새 반의 설치 bat를 실행하세요.

🚫
Chrome 닫아도 자동 복구

Chrome을 강제 종료해도 VBS 감시자가 3~15초 내에 재시작합니다. 완전 제거는 제거 bat 사용.

🔒
설치는 한 번만 실행

설치 bat를 여러 번 실행하면 아이콘이 중복될 수 있습니다. 문제 발생 시 제거 → 설치 순서로 재설치하세요.

👤
계정별 등록

시작 프로그램은 설치를 실행한 Windows 계정에만 등록됩니다. 실제 사용하는 계정으로 설치하세요.

🔧
문제 발생 시 재설치

이상 동작 발생 시 제거 bat → 설치 bat 순서로 재설치하면 대부분 해결됩니다.

🔴
절대 하지 말아야 할 것
  • 한 PC에 여러 반 동시 설치 (예: 설치_1-1.bat 와 설치_1-2.bat 둘 다 실행)
  • 설치 bat를 반복적으로 여러 번 실행
  • VBS 또는 BAT 파일 내용 임의 수정
