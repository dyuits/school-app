📋 Windows 설치 설명서

🔔교실 알림 시스템

교무실에서 학생을 호출하면 교실 전자칠판에 즉시 알림이 표시되는 실시간 교실 알림 시스템입니다.

🪟 Windows 7 이상 🌐 Chrome 필요 📶 인터넷 연결 필요 ⚡ 설치 시간: 1분 이내
🏠 개요 ✅ 요구사항 🚀 설치 방법 🔔 트레이 아이콘 ⭐ 주요 기능 🗑️ 제거 방법 🔧 문제 해결 📁 파일 구조 ⚠️ 주의사항
1

개요 — 교실 알림 시스템이란?

교무실에서 특정 학생을 호출하면 해당 학생의 교실 전자칠판 화면에 큰 알림과 호출음·음성 안내가 즉시 표시됩니다. 수업 중단 없이 조용하고 빠르게 학생을 호출할 수 있습니다.

🏫
교무실 → 교실
교무실 시스템에서 학생을 호출하면 해당 교실 PC로 실시간 전송
📺
전자칠판 전체 알림
화면에 큰 알림 창이 표시되어 수업 중에도 쉽게 확인 가능
🔊
호출음 + 음성 안내
알림음과 함께 학생 이름을 음성으로 안내하여 즉시 인지 가능
⚙️
자동 상주 실행
PC 켜면 자동 시작, 시스템 트레이에 조용히 상주하며 대기
2

요구사항

🪟
Windows 7 이상 Windows 7 / 8 / 10 / 11 모두 지원. PowerShell이 기본 포함되어 있어 별도 설치 불필요.
🌐
Google Chrome 브라우저 Chrome이 설치되어 있어야 합니다. 설치되어 있지 않으면 google.com/chrome 에서 무료로 설치하세요.
📶
인터넷 연결 Firebase 실시간 통신을 사용합니다. 교내 네트워크(유선/무선)에 연결되어 있어야 합니다.
🔑
관리자 권한 불필요 일반 사용자 권한으로 설치 가능합니다. 시작 프로그램은 현재 사용자 계정에만 등록됩니다.
3

설치 방법

1

autostart 폴더 열기

교실 알림 시스템 파일이 담긴 autostart 폴더를 탐색기로 엽니다. 이 설명서 파일(윈도우_설명서.html)이 있는 폴더입니다.

2

해당 반의 설치 파일 더블클릭

학년과 반에 맞는 📄 설치_X-Y.bat 파일을 더블클릭합니다.

예시:

1학년 3반 → 설치_1-3.bat
2학년 7반 → 설치_2-7.bat
3학년 10반 → 설치_3-10.bat
3

설치 완료 확인

검은 명령 창이 잠깐 나타났다가 닫히고, 잠시 후 시스템 트레이(화면 우측 하단 시계 옆)에 교실알림 아이콘이 표시되면 설치가 완료된 것입니다.

💡
트레이 아이콘이 보이지 않을 때

작업표시줄 우측의 ^ (숨겨진 아이콘 표시) 버튼을 클릭하면 아이콘을 확인할 수 있습니다.

4

자동으로 수행되는 작업

설치 bat 파일 실행 시 아래 작업이 자동으로 이루어집니다:

✔ Chrome 앱 모드로 교실 알림 페이지 실행 (작업표시줄에 보이지 않음)
✔ 시스템 트레이 아이콘 등록 (PowerShell 상주 스크립트)
✔ Windows 시작 프로그램에 자동 실행 등록
✔ 스크립트를 %LOCALAPPDATA%\교실알림\ 폴더에 복사하여 보존
4

트레이 아이콘 사용법

화면 우측 하단 시계 옆 알림 영역(트레이)에 교실알림 아이콘이 상주합니다. 이 아이콘을 통해 알림 창을 제어할 수 있습니다.

🖱️
더블클릭
교실 알림 창 보기
👁️
우클릭 → 창 보기
교실 알림 창 표시
🙈
우클릭 → 창 숨기기
교실 알림 창 숨김
❌
우클릭 → 종료
교실 알림 완전 종료
⚠️
Chrome 창의 ✕ 버튼으로 닫아도 자동 재실행됩니다

Chrome 창을 닫아도 트레이 스크립트가 10초 이내에 자동으로 다시 실행합니다. 완전히 종료하려면 트레이 아이콘 우클릭 → 종료를 사용하거나 제거_X-Y.bat 을 실행하세요.

5

주요 기능

🔄
PC 재시작 시 자동 실행 Windows 시작 프로그램에 등록되어 PC를 켜면 자동으로 교실 알림이 시작됩니다.
🔁
Chrome 자동 재실행 Chrome을 닫아도 10초 이내에 트레이 스크립트가 자동으로 재실행합니다.
🔔
시스템 트레이 상주 작업표시줄에 표시되지 않고 트레이에 조용히 상주하여 수업을 방해하지 않습니다.
📢
화면 전체 알림 표시 호출 시 화면 중앙에 큰 알림 창이 팝업되어 수업 중에도 즉시 확인 가능합니다.
🔊
호출음 + 음성 안내 알림음과 학생 이름 음성 안내로 교사와 학생 모두 즉시 인지할 수 있습니다.
⚡
실시간 Firebase 통신 Firebase 실시간 데이터베이스를 통해 교무실 호출이 즉시 교실로 전달됩니다.
6

제거 방법

1

해당 반의 제거 파일 더블클릭

설치한 반에 맞는 🗑️ 제거_X-Y.bat 파일을 더블클릭합니다.

1학년 3반 → 제거_1-3.bat
2학년 7반 → 제거_2-7.bat
2

자동으로 수행되는 작업

✖ 트레이 PowerShell 스크립트 프로세스 종료
✖ 실행 중인 교실 알림 Chrome 창 종료
✖ Windows 시작 프로그램에서 등록 해제
✖ 로컬 데이터 폴더 정리
ℹ️
재설치 시

문제가 생겼을 경우 제거 bat → 설치 bat 순서로 재설치하면 대부분의 문제가 해결됩니다.

7

문제 해결

Q
Chrome이 없다고 나오거나 창이 열리지 않아요
Google Chrome이 설치되어 있지 않습니다.
google.com/chrome 에서 Chrome을 설치한 뒤 설치 bat 파일을 다시 실행하세요.
Q
트레이 아이콘이 보이지 않아요
작업표시줄 우측의 ^ (숨겨진 아이콘 표시) 버튼을 클릭해서 확인하세요.
그래도 없으면 설치 bat를 다시 실행하면 자동으로 트레이 스크립트가 시작됩니다.
Q
소리가 나지 않아요
브라우저의 자동 재생 정책으로 인해 화면을 한 번 클릭해야 소리가 활성화됩니다.
PC 볼륨이 음소거되어 있지 않은지 확인하세요. (작업표시줄 스피커 아이콘)
Chrome 탭의 주소창 옆 🔊 아이콘이 음소거로 표시되어 있으면 클릭하여 해제하세요.
Q
알림 창에 연결 끊김 또는 오프라인 표시가 나와요
인터넷 연결 상태를 확인하세요. 교내 Wi-Fi 또는 유선 LAN에 연결되어 있어야 합니다.
인터넷이 연결되면 자동으로 재연결됩니다. 수동 새로고침이 필요 없습니다.
Q
PC를 재시작했는데 자동 실행이 안 돼요
설치 bat 파일을 한 번 더 실행하면 시작 프로그램이 다시 등록됩니다.
로그인 계정이 달라졌다면 새 계정에서 설치 bat를 실행해야 합니다. (사용자별 등록)
Q
보안 경고 창이 나타나요
Windows Defender SmartScreen 경고가 나타나면 "추가 정보" → "실행"을 클릭하세요.
이 시스템은 학교 내부용 공개 스크립트이며 안전합니다.
Q
호출이 왔는데 알림이 안 보여요
트레이 아이콘을 더블클릭하거나 우클릭 → 창 보기로 알림 창을 띄우세요.
연결 상태가 정상인지 알림 창 하단에서 확인하세요.
문제가 지속되면 제거 후 재설치를 시도하세요.
8

파일 구조

autostart 폴더 안에 있는 파일들입니다. 교실알림.ps1은 수정하지 마세요.

autostart/
  ├─ 교실알림.ps1         ← 트레이 상주 스크립트 (공용, 수정 금지)
  │
  ├─ 설치_1-1.bat          ← 1학년 1반 설치
  ├─ 제거_1-1.bat          ← 1학년 1반 제거
  ├─ 설치_1-2.bat          ← 1학년 2반 설치
  ├─ 제거_1-2.bat          ← 1학년 2반 제거
  ├─ ... (1학년 ~ 3학년, 각 10반)
  │
  ├─ 설치_3-10.bat        ← 3학년 10반 설치
  ├─ 제거_3-10.bat        ← 3학년 10반 제거
  │
  ├─ 윈도우_설명서.html  ← 이 파일
  └─ README.txt           ← 텍스트 요약 안내
📂
설치 후 로컬 복사 위치

설치 bat 실행 시 교실알림.ps1이 C:\Users\[사용자명]\AppData\Local\교실알림\ 폴더에 자동 복사됩니다. autostart 폴더가 이동되어도 정상 동작합니다.

9

주의사항

🏷️
한 PC에 하나의 반만 설치

동일 PC에 여러 반을 설치하면 충돌이 발생합니다. 반드시 하나의 반만 설치하세요.

🔄
반 변경 시 제거 후 재설치

다른 반으로 변경할 경우 먼저 기존 반의 제거 bat를 실행한 후 새 반의 설치 bat를 실행하세요.

🚫
Chrome 닫아도 종료 아님

Chrome 창을 닫으면 자동 재실행됩니다. 완전 종료는 트레이 우클릭 → 종료 또는 제거 bat 사용.

🛡️
ps1 파일 수정 금지

교실알림.ps1 파일을 임의로 수정하면 정상 동작하지 않을 수 있습니다.

👤
사용자 계정별 등록

시작 프로그램은 설치를 실행한 Windows 계정에만 등록됩니다. 공용 PC에서는 실제 사용하는 계정으로 설치하세요.

🔧
문제 발생 시 재설치

알 수 없는 문제 발생 시 제거 bat → 설치 bat 순서로 재설치하면 대부분 해결됩니다.

🔴
절대 하지 말아야 할 것
  • 한 PC에 여러 반 동시 설치 (예: 설치_1-1.bat 와 설치_1-2.bat 둘 다 실행)
  • 교실알림.ps1 파일 내용 임의 수정
  • %LOCALAPPDATA%\교실알림 폴더 임의 삭제 (제거 bat 사용 권장)
