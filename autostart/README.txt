==========================================
  교실 알림 자동 설치 안내 (Windows PC)
==========================================

[설치 방법]
1. 해당 반의 "설치_X-Y.bat" 파일을 더블클릭합니다.
   (예: 1학년 3반 → 설치_1-3.bat)

2. 자동으로 다음이 수행됩니다:
   - 시스템 트레이 (알림 영역)에 아이콘 등록
   - Chrome 앱 모드로 교실 알림 페이지 실행 (숨김 상태)
   - Windows 시작 프로그램에 자동 실행 등록

3. 주요 특징:
   - 시스템 트레이 (시계 옆)에 아이콘이 표시됩니다.
   - Chrome 창이 작업표시줄에 보이지 않습니다 (완전 숨김).
   - 호출이 오면 Chrome 창이 자동으로 나타나고 소리가 재생됩니다.
   - 소리가 사용자 인터랙션 없이도 자동 재생됩니다.
   - Chrome을 닫아도 (X 버튼) 10초 내에 자동으로 다시 실행됩니다.
   - PC를 재시작해도 자동으로 시작됩니다.

4. 트레이 아이콘 사용법:
   - 더블클릭: 교실 알림 창 보기
   - 우클릭 → "창 보기": 교실 알림 창 표시
   - 우클릭 → "창 숨기기": 교실 알림 창 숨김
   - 우클릭 → "종료": 교실 알림 완전 종료

[제거 방법]
1. 해당 반의 "제거_X-Y.bat" 파일을 더블클릭합니다.
   (예: 1학년 3반 → 제거_1-3.bat)

2. 자동으로 다음이 수행됩니다:
   - 트레이 스크립트 (PowerShell) 종료
   - 실행 중인 교실 알림 Chrome 종료
   - 시작 프로그램 등록 해제 (VBS 파일 삭제)

[요구사항]
- Google Chrome이 설치되어 있어야 합니다.
- Windows PowerShell이 필요합니다 (Windows 7 이상 기본 포함).
- 인터넷 연결이 필요합니다 (Firebase 실시간 통신).

[주의사항]
- 한 PC에 하나의 반만 설치하세요.
- 문제 발생 시 제거 후 다시 설치해주세요.
- 완전히 종료하려면 반드시 "제거_X-Y.bat"을 사용하세요.
  (Chrome을 닫아도 트레이 스크립트가 자동으로 다시 실행합니다.)
- 트레이 아이콘이 보이지 않으면 작업표시줄의 ^ (숨겨진 아이콘 표시)를 
  클릭하여 확인하세요.

[파일 구조]
autostart/
  교실알림.ps1          ← 트레이 스크립트 (공용, 수정하지 마세요)
  설치_X-Y.bat          ← 설치 (더블클릭)
  제거_X-Y.bat          ← 제거 (더블클릭)
  README.txt            ← 이 파일
