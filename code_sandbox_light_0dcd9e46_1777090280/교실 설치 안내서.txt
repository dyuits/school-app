========================================
  📢 교실 알림 시스템 설치 안내서
========================================

■ 설치 방법

  [방법 1] Windows PC — 자동 설치 (추천)

  ① "autostart" 폴더를 교실 PC에 복사
  ② 해당 반의 "설치_X-Y.bat" 더블클릭
     예시: 1학년 3반 → 설치_1-3.bat
  ③ 끝! 자동으로 실행됩니다.

  특징:
  - 시스템 트레이 (시계 옆)에 아이콘 표시
  - Chrome 창이 완전히 숨김 상태로 동작
  - 호출 시 자동으로 창 표시 + 소리 재생
  - 소리가 터치 없이 자동 재생 (autoplay 정책 우회)
  - PC 재시작 시 자동 실행
  - Chrome을 닫아도 10초 내 자동 재실행
  - 트레이 아이콘 더블클릭: 창 보기
  - 트레이 아이콘 우클릭: 창 보기/숨기기/종료
  - 완전 종료: "제거_X-Y.bat" 더블클릭


  [방법 2] 안드로이드 전자칠판 — GitHub Pages

  ① Chrome에서 아래 주소 접속
     https://dyuits.github.io/school-app/classroom.html
  ② 해당 반 선택 (또는 URL에 ?class=1-3 추가)
  ③ 화면을 한 번 터치하여 소리 활성화
  ④ (선택) Chrome 메뉴(⋮) → "홈 화면에 추가" → 앱처럼 사용

  QR 코드 인쇄:
  - https://dyuits.github.io/school-app/qr.html 접속
  - 30개 반 QR 코드를 한 번에 인쇄 가능


■ 사용 방법

  - 교무실에서 학생을 호출하면
  - 교실 화면에 자동으로 크게 표시됨
  - 띵동 소리 + 음성으로 읽어줌
  - 끝나면 자동으로 대기 화면으로 복귀


■ 설정 변경

  교실 화면 좌측 상단 ⚙️ 설정 버튼:
  - 음성 선택 (목소리 변경)
  - 호출음 변경 (딩동, 차임, 멜로디 등 8종)
  - 호출 횟수 (1~3회)
  - 볼륨 / 음성 크기 조절
  - 소리 테스트 / 음성 테스트 버튼으로 확인


■ 문제 해결

  Q: "❌ 연결 끊김"이 표시돼요
  A: 인터넷 연결을 확인하세요. 와이파이가 연결되어 있어야 합니다.

  Q: 소리가 안 나요
  A: [Windows PC] Chrome 플래그로 자동 재생이 활성화되어 있어
     별도 터치 없이 소리가 나야 합니다. 안 나면 제거 후 재설치하세요.
     [전자칠판] 화면을 한 번 터치하여 소리를 활성화하세요.
     ⚙️ 설정에서 볼륨을 올리고, "소리 테스트" 버튼을 눌러보세요.
     브라우저 음소거가 아닌지 확인하세요.

  Q: 화면이 잠겨요 (절전모드)
  A: Windows: 설정 → 전원 → 화면 끄기 "안 함" 설정
     Android: 설정 → 디스플레이 → 화면 자동 꺼짐 "없음"

  Q: Windows에서 완전히 종료하고 싶어요
  A: "제거_X-Y.bat" 더블클릭 (Chrome 닫기만으로는 종료 안 됨)


■ 준비물
