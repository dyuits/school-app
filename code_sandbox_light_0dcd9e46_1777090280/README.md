# 제주중앙고등학교 스마트 수업교체 도우미 v3.1

## 프로젝트 개요
교사 시간표 기반 수업 교체·대체 후보를 자동으로 찾아주는 웹 애플리케이션입니다.

## 완료된 기능

### 핵심 기능
- ✅ **수업교체 탭**: 전체 교사 시간표 표시, 수업 셀 클릭 시 교체/대체 후보 자동 검색
- ✅ **교사 시간표 탭**: 개별 교사 상세 시간표 조회 (학년군, 공강 수, 담임 표시)
- ✅ **공강 현황 탭**: 각 교사별 공강 시간 목록 표시
- ✅ **협의회 탭**: 여러 교사 동시 선택 → 공통 공강 시간 찾기
- ✅ **학사일정 탭**: 2026학년도 월별 학사일정 달력 표시
- ✅ **교직원 연락처 탭**: 부서별/교과별 교직원 연락처 (잠금 해제 필요)
- ✅ **학급 명단 탭**: 학년/반별 수업 담당 교사 명단
- ✅ **강의실 현황 탭**: 특별실 사용 현황
- ✅ **교체 불가 시간 설정**: 교사별 특정 시간 교체불가 설정 (잠금 해제 필요)

### 색상 규칙
| 색상 | 의미 | 교체 | 대체 |
|------|------|------|------|
| **노란색** | 선택과목 | ❌ 불가 | ✅ 가능 |
| **민트색** | 시간강사·산학협력교사 | ❌ 불가 | ❌ 불가 |
| **초록색** | 온라인·체육순회 수업 | ❌ 불가 | ❌ 불가 |
| **하늘색** | 선택된 수업 | - | - |
| **연두색** | 교체 가능 후보 | - | - |

### 특수 처리 교사
- **시간강사**: 이순규(종교), 강진석(음악), 강소희(화법) → 민트 표시
- **산학협력교사**: 이상분(컴그) → 민트 표시
- **체육순회**: 수요일 전담 체육 순회 → 초록 표시, 교체·대체 불가
- **온라인/물리온라인**: 교사 없는 온라인 수업 → 초록 표시, 교체·대체 불가

## 데이터 구조

### TEACHER_SCHEDULE
```js
{ 교사명: { 요일교시: "반번호 과목명", ... } }
// 예: { "황혜인": { 월1: "106 국어1", ... } }
// 반번호: 101~110 (1학년), 201~210 (2학년), 301~310 (3학년)
// 311이상: 선택과목 (노란색)
```

### MINT_TEACHERS
```js
new Set(["이순규", "강진석", "강소희", "이상분", "체육순회", "온라인", "물리온라인"])
```

### SUBJECT_SUBSTITUTE_MAP
```js
{ 교사명: { subject: "교과명", canSubFor: ["대체가능교사1", ...] } }
```

### HOMEROOM_TEACHERS
```js
{ "1-1": "강부열", "1-2": "강혜민", ... }
```

## 아직 구현/확인이 필요한 것
- 일부 교사 시간표 데이터 미확인 (OCR 오류 가능성)
- 홍민영 선생님 정확한 과목명 재확인 필요 (A_미술, 실무 등)
- 오재영 선생님 정확한 데이터 재확인 (수학 vs 과학 혼재)
- 선택과목 이름 정확도 검증 (A_조각, E_문학 등 영문 접두어)
