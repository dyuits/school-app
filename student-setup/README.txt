==========================================
  교실 알림 - 학생 PC 설치 안내
==========================================

[설치 방법]
1. 이 폴더(student-setup)를 PC 아무 곳에나 압축 해제합니다.
   예) C:\교실알림\

2. autostart\ 폴더 안에서 해당 반의 설치 파일을 더블클릭합니다.
   예: 1학년 3반  → 설치_1-3.bat
       2학년 7반  → 설치_2-7.bat

3. 설치 완료 후 트레이(시계 옆 ^ 아이콘)에 아이콘이 생깁니다.

[제거 방법]
autostart\ 폴더 안에서 해당 반의 제거 파일을 더블클릭합니다.
   예: 1학년 3반  → 제거_1-3.bat

[요구사항]
- Google Chrome 설치 필요
- Windows PowerShell (Windows 7 이상 기본 포함)
- 인터넷 연결 필요 (Firebase 실시간 통신)

[주의사항]
- 한 PC에 하나의 반만 설치하세요.
- 문제 발생 시 제거 후 재설치하세요.
- autostart\ 와 classroom\ 폴더는 반드시 같은 위치에 있어야 합니다.

[폴더 구조]
student-setup\
  autostart\    <- 설치/제거 bat 60개 + 교실알림.ps1
  classroom\    <- 교실별 HTML 30개
  README.txt    <- 이 파일
