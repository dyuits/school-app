배포 후 사용 순서

  • GitHub Pages를 켠 뒤 `docs/` 폴더를 배포 대상으로 설정합니다.
  • 전자칠판에서 `classroom.html?class=학년-반` URL로 접속합니다.
  • 한 번 터치해 전체 화면과 오디오를 활성화합니다.
  • Chrome 메뉴에서 홈 화면에 추가하여 앱처럼 실행합니다.
