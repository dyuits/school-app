전자칠판 설치 안내

안드로이드 전자칠판에서 GitHub Pages 주소만 열면 교실 알림을 사용할 수 있습니다. 초기 설정은 1~2분이면 끝납니다.

교실 알림 열기 QR 코드 인쇄

1Chrome에서 URL 접속

  1. 전자칠판에서 Chrome 브라우저를 실행합니다.
  2. GitHub Pages 주소 또는 반별 QR 코드를 엽니다.
  3. 예: `classroom.html?class=1-1` 또는 `classroom.html` 후 반 선택

2반 선택 및 초기 권한 허용

  1. 반 파라미터가 없으면 1-1부터 3-10까지 버튼이 표시됩니다.
  2. 해당 반을 선택하면 교실 대기 화면이 열립니다.
  3. 화면을 한 번 터치해 전체 화면, 오디오, 화면 켜짐 유지 기능을 활성화합니다.

3홈 화면에 추가

  1. Chrome 우측 상단 메뉴를 엽니다.
  2. `홈 화면에 추가` 또는 `앱 설치` 메뉴를 선택합니다.
  3. 앱 이름을 `교실 알림`으로 저장합니다.
  4. 이후 홈 화면 아이콘을 눌러 앱처럼 실행합니다.

브라우저 종류에 따라 메뉴 이름은 조금 다를 수 있습니다.

4전체 화면과 절전 방지

  1. 페이지를 처음 터치하면 전체 화면 전환을 시도합니다.
  2. Wake Lock 지원 기기에서는 화면 꺼짐 방지를 자동 요청합니다.
  3. 기기 정책상 잠금이 계속되면 안드로이드 설정에서 자동 꺼짐 시간을 길게 설정하세요.

운영 팁

  • 좌측 상단 설정에서 음성, 호출음, 볼륨, 반복 횟수를 반별로 저장할 수 있습니다.
  • 상단 연결 표시가 `✅ 연결됨`이면 Firebase 실시간 수신이 정상입니다.
  • 오프라인 또는 연결 끊김 시 자동으로 재연결을 시도합니다.
  • QR 코드를 출력해 전자칠판 옆에 붙여두면 재설치가 쉬워집니다.
